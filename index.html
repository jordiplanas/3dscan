¿Qué es la
fotogrametría?

La fotogrametría es una técnica para crear modelos 3D con texturas foto-realistas. Se toman centenares de imágenes de distintos puntos de vista de un producto y se procesan en un software para obtener una replica en tres dimensiones. Esto se realiza en un plató equipado con un gran número de cámaras dispuestas para realizar una captura envolvente 360º del objeto.

Previous Next

Una parte del proceso está automatizado para mayor fluidez, el disparo de las cámaras y la descarga de las imágenes. Para el postprocesado contamos con un artista digital que retoca el resultado para asegurar la máxima calidad.

Los scans en alta resolución se usan para crear imágenes de producto y videos que requieran gran detalle. A partir de este modelo High se extraen otras versiones optimizadas para Web, Apps, AR (Realidad Aumentada) o VR (Realidad Virtual).

La fotogrametría ofrece mayor rapidez, escalabidilidad y versatilidad respecto a otros procesos de scan. Ofrece acabados hiper-realistas en modelos complejos y orgánicos.
